3dsMax WebGL Exporter - Installation and Use

Installing the Plugin

Copy webgl.dle and webgl.html to Max's 'stdplugs' folder.

The Max installer makes a 3dsMax folder in 'My Documents' which contains, among many other things, an 'export' folder, which is the default target folder for all exporters. Your output will go there and will need to be copied to the Previewer's 'exports' folder is to see it. See below for Previewer instructions.

Setting Up Your Models for Export

  • Make sure to use Standard Materials only. No hierarchy.
  • Avoid transparency where mesh edges are not 'clean' (things built for render-only can be sloppy that way).

Exporting a Model

  1. On the 'file' menu (well there is no file menu, it's the rather large logo icon top-left of the window), choose Export.
  2. From the list of export file types, choose WebGL (.html).
  3. Enter a file name ending in .html, and hit OK. This will generate a folder of that name without the extension, and put an html and JavaScript file of that name in the new folder.
  4. To preview the exported file, copy the new folder in its entirety to the 'exports' folder of the Previewer application (see below).

The Previewer

The Previewer is a web application that uses Three.js to render the JSON model files. It requires a running web server to operate. To set it up:
  1. Copy the entire contents of the 'previewer' folder to somewhere your web server.
  2. Make an empty folder called 'exports' at the same level as 'previewer.'
  3. To preview an exported file, copy the folder, in its entirety, from the 3dsMax 'export' folder into the 'exports' folder on the web server.
  4. If you're not sure how the folder structure should be laid out, refer to the top-level folder of this project and have a look at the sample exports.

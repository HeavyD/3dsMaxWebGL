3dsMax WebGL Exporter - Building the Plugin

Building the Plugin

  1. Open Visual C++ 2010 (any version; Express OK too).
  2. Drop the webgl2012 folder into the SDK's 'samples/import_export' folder. (it will be at peer level with asciiexp and vrmlexp).
  3. Open 'samples' solution.
  4. Select Add Project, choose the webgl2013 project file.
  5. Choose your configuration (Debug, Release etc.)
  6. Build
  7. webgl.dle file is deposited into Max's stdplugs folder.
    • 64 bit under "Program Files Max folder"
    • 32 bit under "Program Files (x86) Max folder"
